AnimationCheckTools

AnimationCheckTools is a tool that checks animation files for issues with a single click, without a user interface. The main checks include whether there is a camera in the scene, whether there are unloaded reference files, and whether the camera is locked. It also adds the first frame to the animation file, removes unused layers, and clears unknown nodes.
