TrimmerTools

TrimmerTools is a tool for editing audio or video based on an EDL



1.Copy the Trimmer.exe to your own drive, and please use an English path for the copy.



2.Select the file type to edit (audio or video),



3.choose the EDL and editing file, and the output folder, then press RUN to execute.

Note!!!

1.Please use English paths for input and output!

2.When editing videos, avoid interrupting the process. If you need to interrupt (by closing the black command prompt window), you will need to delete the partially trimmed video and start the process again from the beginning.
